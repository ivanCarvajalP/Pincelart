Estas reglas bloquean TODO excepto lectura autenticada.
Tu sitio web NO funcionará público - solo con login.

REGLA FIRESTORE:
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /productos/{productId} {
      allow read: if request.auth != null;  // Solo autenticados
      allow write: if false;
    }
    match /usuarios/{userId} {
      allow read, write: if false;
    }
    match /ventas/{ventaId} {
      allow read, write: if false;
    }
    match /{document=**} {
      allow read, write: if false;
    }
  }
}

⚠️ PROBLEMA: Tu sitio web público NO podrá mostrar productos
⚠️ NECESITAS: Sistema de login obligatorio para visitantes
